
# Wall of Shame

A website where you can display you worst blunders in the form of code.

Users don't require a login. A userID is generated off of the browser and that acts as the basic auth.




## Working

Basically a wall of card with snippets of not-so-proud code that you've written. 
Anyone can view and add comments to it. Only the user who has made the commit can delete it.

## Tech Stack

- Node
- React (react-hot-toast + daisyUI)
- Express
- MongoDB
- Vite
- TailwindCSS
- redis



## How to run

Make sure 
- [Node.js](https://nodejs.org/) is installed
- redis is set-up on upstash
- mongodb is set up 


**1. Clone the repo**
```bash
git clone <repo-url>
```

**2. Start backend**
```bash
cd backend 
npm run dev
```
**3. Start frontend**
```bash
cd frontend 
npm run dev
```
**4. Run**

    Go to http://localhost:5173
